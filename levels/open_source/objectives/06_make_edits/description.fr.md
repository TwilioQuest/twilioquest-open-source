# Laisser sa marque

Maintenant, la partie amusante&nbsp;: apporter des modifications au projet&nbsp;! Pour ouvrir ce coffre et revendiquer la __Flamme de l'Open Source__, vous devez modifier le code source dans le projet `open-pixel-art` et __valider vos modifications__ dans votre branche de développement. Reportez-vous à la section «&nbsp;Aide&nbsp;» pour obtenir des instructions sur la manière d'effectuer votre modification.

Une fois que vous avez __validé vos modifications__ dans votre branche de développement, `<%= env.TQ_OPEN_PIXEL_ART_BRANCH.value %>`, cliquez sur le bouton *HACK*.
